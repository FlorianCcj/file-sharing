# file-sharing

## Introduction
This project has been created to share big file with other people fastly.

### file-sharing-back
nodejs with express 4.16.2 and multer 1.3.0.

### file-sharing-front
angular 5.2.5

## dev
to work on this project you need node (I use v9.4.0) and npm (use 5.6.0)

### file-sharing-back
install: npm i

start: npm start

### file-sharing-front
install : npm i

start: npm start
